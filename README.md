# My Website

A personal website hosted on GitHub Pages.

## Setup

1. Push this repository to GitHub
2. Go to your repository settings
3. Navigate to "Pages" in the left sidebar
4. Under "Source", select the branch you want to deploy (usually `main` or `master`)
5. Your site will be available at `https://yourusername.github.io/mynpco.github.io/`

## Customization

- Edit `index.html` to change the content and structure
- Modify `styles.css` to customize the appearance
- Update `script.js` to add interactivity

## Features

- Responsive design
- Smooth scrolling navigation
- Modern, clean UI
- Ready for customization
